已读乱码 020 | 学前端，写文档

15 Jul, 2025
|Edit page

虽然说是这周，实际上已经是周一了，现在时间是周二的凌晨，一直拖着不写，拖着拖着就拖到现在这个点。

前端，还是前端

我学前端的过程还是蛮坎坷的。最开始就是学 HTML, CSS, JavaScript 三件套，实际上也一直没学明白，只是学习了个皮毛，以至于之前 CSS 连 flex 和 grid 布局我都还没学明白，就想着去看 Tailwind CSS，简直是自讨苦吃。后来就是顺着技术栈往下学习，TypeScript, React, Next.js，这些技术我都是反复的学，就没有能够一次学完的。不过技术上我也不是很推崇一次学完这种方式，一次吃透那要学到猴年马月。学会一点皮毛之后，就可以学中做，做中学，这才是比较好的学习方式。

Next.js 的 React 和 Next.js 本身的学习教程我觉得还是比较优质的文档，基本上前端开发里很基础的点都有提及，一整个周末的时间也足够看完。我自己做完 Next.js 的学习教程的链接在这里。

留痕

我自己是有些记录笔记的习惯的，但是实际没那么多。很多概念都是当时清楚，过一段时间就忘记，笔记还是要记的。我的本地笔记是用的 Obsidian，没有使用云同步的功能，单纯作为本地笔记来用。功能上来说，Obsidian 足够用了，但也因为是本地笔记，不便于分享，大部分时间只有我自己在看，我觉得我记录的技术相关的笔记还是有分享的价值的，公开比较合适，就在 Vercel 上部署了一个笔记网站，本来是挂在我的主域名上的，现在是放在 doc.theknot.dev，我觉得还是值得一看的。不过现在这个时间点，文档很不完善，也只有一点点，我会慢慢补充我现在记录的和以后记录的技术知识，用作留痕。

开发帐号

上面的 theknot.dev 域名是我最近才注册的，我最近注册了开发相关的帐号，和我的个人帐号完全分隔开来，也许以后做点什么就都放在单独的开发帐号上。

最后

虽然说是要慢慢来，还是开始做吧。
