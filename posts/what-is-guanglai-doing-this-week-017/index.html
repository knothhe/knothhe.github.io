这次的周刊大概也是我写得最晚的一次了，拖到了下一周的周一。不过也得得益于昨天没有在很晚的时候还写周刊，昨天我的睡眠在很多的周日到周一的晚上里是睡眠很不错的一次。周一上班上班前的那个晚上真的是太容易失眠了。

上海野生动物园

我也算是第二次去上野了，上次去还是去年十月份和小红书上的网友一起去的，当时也算是第二次见面，然后也是最后一次见面。所以我对于动物园也有一些不太好的记忆，不过无所谓了，我对海洋馆也有不好的记忆，这些适合约会的场所，我之后可能都会有些不美好的记忆，我总不至于因为这个原因这些地方就不去了吧。

这次逛动物园天气不算太好，阴天，气温还很低。上海在清明节前的天气总是那么反复，整个三月，在高温天气开始，气温一路走低，上周末罕见又高温一次，这周末气温又再次走低。反复无常的气候，阴郁的心情。

和她们俩逛动物园也是真的迅速，她们的腿脚比我的利索多了，十点多进动物园，下午一点多就逛完了动物园，我原本的设想是大概下午能逛到晚饭的时间，中午在动物园里吃顿午饭的，她们这个逛的速度完全超过了我的预期，于是也只能在逛玩之后，带着她们两个人去了长泰广场吃了下午饭。这次逛动物园，也有逛了些上次没有逛到的地方。上次是没有去猛兽区的，这次也是去猛兽区逛了一圈，拍下了不少照片。不过整体逛动物园的体验也确实如其他人所说，没有那么好，除去门票的 120¥，动物园内部很多需要互动的项目都是需要单独收费的，有看到 30¥，也有看到 60¥的项目，猛兽区单独买票则是需要花费 50¥。好处则是和动物的互动性确实很多，狐猴岛更是直接开放，可以和狐猴近距离接触，也是我最喜欢的地方。

和不同的人一起逛的感觉总是不太一样，上次和网友一起逛的时候，天气还是很好的，阳光明媚。整体逛起来也是不急不慢，慢慢闲逛，总体上也有在聊天，互相了解。

上野动物园就到此为此，下次估计会和朋友去上海动物园，门票便宜点，似乎也没有那么多额外的收费项目。

新玩具

周末买的新玩具，Z502 相机也总算在周三到手。中年人的三大爱好，相机，钓鱼，充电头，我也是有了其一。相机周三到手，周六就带去动物园拍照。我是买的相机主题加上 ZDX 16-50 套机镜头和 ZDX 50-250 镜头。一开始我使用 50-250 镜头拍照的，但是这颗镜头只能对焦近处的物体，我预估大概是 2m，再远点的地方就一点都对焦不上。我一直以为是我的问题，在动物园也没什么时间调整，她们走得实在是太快了，后来就一直使用 16-50 拍照。整体上还是有点耗电的，大概在猛兽区的时候，相机就快没电了，好在当时也买了 jcc 的 typec 充电替换电池，立马换上满电的电池。

周日下午我又尝试了下 50-250 镜头，发现确实是有些问题的，近处的画面能对焦成，远处的无论如何都对焦不上。我也有上网搜索是否有类似情况，是否有解决办法，结果让我搜索到了一些古早的帖子，大概最后都是猜测是镜头的问题。我最后不得已，还是只能联系淘宝客服。不过淘宝客服也只是让我各种尝试调整对焦类型之类的参数，然而当然是毫无效果。第一次买相机就碰到镜头的问题，也真实让我很头疼，小白难受。最后就是在写文章之前，和客服聊好，申请换货镜头，把镜头发回给他们看看有什么问题。

拍照的整体效果我是很满意的，整个周内我也断断续续看了不少相机拍照的视频。曝光三要素，快门，光圈，ISO 也是牢牢记在心里，什么景深，虚化，透视也多多少少了解了一些，构图则应该是需要很长时间学习的，我还没有很懂，以后慢慢总结。总体上，拍照好难啊。

我还尝试下载 photoshop 和 lightroom 学习修图，找到了开源的破解工具，这里是在 GitHub 上的地址，作者倒是真的有点儿有趣的。

修图也要慢慢学起来了。

最后

上周也是因为似乎没有什么可以记录的事情，所以一拖再拖，拖到此刻才写完。那上周的周刊，已读乱码，就到此为此。
